## Component Library

**Library**: shadcn/ui *(recommended — Claude knows it deeply, components live in your codebase)*
**Tailwind CSS**: required
**Docs**: https://ui.shadcn.com

```bash

npx shadcn@latest init

# Add components as needed
npx shadcn@latest add button card table dialog form input
```

**Primitive components location**: `src/components/ui/` *(auto-generated by shadcn, do not edit)*
**Composed components location**: `src/components/` *(project-specific, built on top of primitives)*

## Color Palette

> Colors are defined as CSS variables in `src/app/globals.css` (Next.js) or `src/index.css`.
> Claude must use Tailwind semantic classes — never raw hex values, never arbitrary colors.

| Token                  | Tailwind class             | Usage                              |
| ---------------------- | -------------------------- | ---------------------------------- |
| `--background`         | `bg-background`            | Page background                    |
| `--foreground`         | `text-foreground`          | Default body text                  |
| `--primary`            | `bg-primary`               | Primary buttons, key CTAs          |
| `--primary-foreground` | `text-primary-foreground`  | Text on primary color              |
| `--secondary`          | `bg-secondary`             | Secondary buttons, chips           |
| `--muted`              | `bg-muted`                 | Subtle backgrounds, disabled states|
| `--muted-foreground`   | `text-muted-foreground`    | Placeholders, captions, metadata   |
| `--accent`             | `bg-accent`                | Hover backgrounds                  |
| `--destructive`        | `bg-destructive`           | Errors, delete actions             |
| `--border`             | `border-border`            | All borders and dividers           |
| `--ring`               | `ring-ring`                | Focus rings                        |
| `--card`               | `bg-card`                  | Card backgrounds                   |

### Rules
- ✅ Use CSS variable tokens: `bg-primary`, `text-muted-foreground`, `border-border`
- ✅ Use semantic Tailwind: `text-destructive`, `bg-accent`
- ❌ Never raw hex: ~~`#3b82f6`~~
- ❌ Never arbitrary Tailwind colors: ~~`bg-blue-500`~~ unless you add it to the palette above

## Spacing & Layout

**Page wrapper**: `max-w-7xl mx-auto px-4 sm:px-6 lg:px-8`
**Sidebar width**: [e.g., 240px]
**Breakpoints**: mobile-first — `sm` (640px), `md` (768px), `lg` (1024px), `xl` (1280px)

| Scale  | Tailwind | px   | Use for                         |
| ------ | -------- | ---- | ------------------------------- |
| Tight  | `gap-1`  | 4px  | Icon + label pairs              |
| XS     | `gap-2`  | 8px  | Inline items, tight lists       |
| S      | `gap-3`  | 12px | Form field internals            |
| Base   | `gap-4`  | 16px | Default spacing between elements|
| M      | `gap-6`  | 24px | Between sections in a card      |
| L      | `gap-8`  | 32px | Between major sections          |
| XL     | `gap-12` | 48px | Between page-level sections     |

**Rule**: Only use Tailwind spacing classes. Never `p-[13px]` or `mt-[22px]`.

---

## Shape & Elevation

**Default border radius**: `rounded-lg` *(pick one and never mix)*
**Card shadow**: `shadow-sm` for cards, `shadow-md` for popovers, `shadow-lg` for modals

## Non-Negotiable Rules

1. **Read this file first** — before writing a single line of UI code
2. **Check Component Patterns** — match existing patterns exactly before inventing new ones
3. **Check Component Inventory** — don't build what already exists
4. **Library primitives first** — always use shadcn components before building custom
5. **Palette only** — no colors outside the token table above
6. **Spacing scale only** — no arbitrary spacing values
7. **Mobile-first** — every component works on mobile before adding `md:` / `lg:` variants
8. **Accessible** — every interactive element has an aria-label, role, or visible label; respect keyboard navigation
9. **Update this file** — when you establish a new pattern, add it to Component Patterns and Component Inventory
